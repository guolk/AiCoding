## 1. 测试概述

### 1.1 测试目的

确保应用在不同操作系统上的行为符合预期，特别是：
- **macOS**: 关闭窗口后应用保持运行，截图支持 Retina 屏幕
- **Windows**: 关闭窗口后应用正常退出
- **Linux**: 与 Windows 行为一致

### 1.2 测试环境要求

| 平台 | 最低版本 | 推荐版本 |
|------|----------|----------|
| Windows | Windows 10 1909 | Windows 11 22H2 |
| macOS | macOS 10.15 (Catalina) | macOS 14 (Sonoma) |
| Linux | Ubuntu 20.04 | Ubuntu 22.04 |

### 1.3 硬件要求

- **Retina 屏幕测试**: MacBook Pro 或其他高 DPI 显示器 (2x 缩放)
- **多显示器测试**: 至少 2 台显示器
- **普通屏幕测试**: 标准 1080p 显示器 (1x 缩放)
