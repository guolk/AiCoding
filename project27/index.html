使用帮助

基本操作

  • • 创建节点: 点击工具栏中的节点类型按钮
  • • 创建连接: 从输出端口(绿色)拖拽到输入端口(红色)
  • • 移动节点: 点击并拖拽节点
  • • 平移画布: 按住 Alt + 鼠标拖拽，或鼠标中键拖拽
  • • 缩放画布: 鼠标滚轮

选择操作

  • • 选择单个: 点击节点或连接线
  • • 多选: 按住 Shift 点击添加到选择
  • • 全选: Ctrl + A
  • • 取消选择: Esc 或点击空白处

编辑操作

  • • 删除: 选中后按 Delete 或 Backspace
  • • 撤销: Ctrl + Z
  • • 重做: Ctrl + Y 或 Ctrl + Shift + Z

小地图导航

  • • 点击右下角小地图可快速跳转到指定位置
  • • 橙色虚线框表示当前视口范围

节点类型

  • • 矩形: 通用节点，用于表示流程步骤
  • • 菱形: 用于表示判断/决策
  • • 圆形: 用于表示开始/结束
  • • 平行四边形: 用于表示输入/输出
  • • 泳道: 用于分组和组织其他节点
