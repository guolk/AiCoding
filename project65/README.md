# 马拉松训练计划与跑步追踪工具

一个功能完整的在线马拉松训练管理工具，帮助跑者科学训练、健康完赛。

## 功能模块

### 1. 训练计划模块
- **目标赛事设置**：设置比赛日期、目标成绩，自动计算剩余训练周数
- **个性化计划生成**：根据当前跑量和目标，生成12-24周的系统训练计划
- **计划灵活调整**：因伤或生活安排调整单次训练后，自动重排后续计划

### 2. 跑步记录模块
- **数据录入**：记录每次跑步的日期、距离、时间、配速、心率、感受
- **GPX轨迹导入**：支持从Garmin、苹果手表等导出的GPX文件导入，可视化展示跑步路线
- **跑步类型分类**：轻松跑、节奏跑、间歇训练、长距离慢跑、比赛

### 3. 数据分析模块
- **周跑量趋势图**：柱状图展示每周公里数，对比计划和实际完成情况
- **配速进步曲线**：相同距离的配速历史趋势，直观展示训练效果
- **心率区间分布**：各心率区间训练时间占比分析，优化训练强度

### 4. 比赛准备模块
- **赛前减量计划**：自动生成赛前2周的减量安排
- **配速计划器**：输入目标成绩，生成分段配速策略（支持平坦/起伏/多坡赛道）
- **比赛检查清单**：装备、营养、交通的准备事项，确保万无一失

### 5. 伤病管理模块
- **伤病记录**：记录伤病类型、严重程度、症状和康复建议
- **训练替代方案**：针对常见伤病提供保守训练建议
- **训练负荷预警**：跑量增加过快时自动提醒（遵循10%原则）

## 技术栈

- **前端框架**：React 18 + TypeScript
- **构建工具**：Vite 5
- **图表库**：Chart.js + react-chartjs-2
- **地图组件**：Leaflet
- **日期处理**：date-fns
- **数据存储**：LocalStorage（浏览器本地存储）

## 数据安全

所有数据保存在浏览器本地存储（LocalStorage）中，不会上传到任何服务器。请定期导出重要数据进行备份。

## 训练原则

1. **循序渐进**：周跑量增幅不超过10%
2. **劳逸结合**：保证充足的恢复时间
3. **Listen to your body**：注意身体信号，疼痛时及时休息
4. **交叉训练**：结合游泳、自行车等低冲击运动
5. **力量训练**：加强核心和下肢力量，预防伤病
