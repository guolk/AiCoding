## 📱 API 接口测试

可以使用Postman或curl测试以下API接口：

### 书籍相关
```bash
# 获取所有书籍
GET http://localhost:3001/api/books

# 获取单本书籍详情
GET http://localhost:3001/api/books/{bookId}

# 通过ISBN添加书籍
POST http://localhost:3001/api/books/isbn/9787020024759

# 更新书籍状态
PATCH http://localhost:3001/api/books/{bookId}/status
Content-Type: application/json
{ "status": "reading" }
```

### 阅读会话相关
```bash
# 开始阅读会话
POST http://localhost:3001/api/reading/sessions
Content-Type: application/json
{ "bookId": "{bookId}", "startPage": 0 }

# 结束阅读会话
PATCH http://localhost:3001/api/reading/sessions/{sessionId}/end
Content-Type: application/json
{ "endPage": 50 }
```

### 笔记相关
```bash
# 获取书籍笔记
GET http://localhost:3001/api/books/{bookId}/notes

# 创建笔记
POST http://localhost:3001/api/notes
Content-Type: application/json
{
  "bookId": "{bookId}",
  "type": "highlight",
  "content": "这是一段精彩的摘录",
  "pageNumber": 42,
  "chapter": "第一章",
  "topics": ["哲学", "人生"]
}
```

### 书单相关
```bash
# 获取所有书单
GET http://localhost:3001/api/reading-lists

# 获取相似书籍推荐
GET http://localhost:3001/api/books/{bookId}/similar

# 获取年度报告
GET http://localhost:3001/api/reading/annual-report/2024
```

## ✨ 特色功能演示

### 1. 智能阅读速度分析
系统根据历史阅读数据计算：

### 3. 书单分享与导入
- 生成私密/公开的分享链接
- 支持一键导入他人分享的书单
- 可以对导入的书单进行二次编辑

### 4. Open Library 集成
- 通过ISBN自动获取完整书籍元数据

## 🔧 常见问题排查

### 问题1: 前端无法连接后端
**解决方案**:
- 确认后端服务器运行在端口3001
- 检查浏览器控制台是否有CORS错误
- 刷新页面重试

### 问题2: ISBN搜索无结果
**解决方案**:
- 确认ISBN格式正确（10位或13位）
- Open Library API可能有延迟，重试一次
- 可以手动添加书籍信息

### 问题3: 数据库连接失败
**解决方案**:
- 检查library.db文件是否存在
- 确认文件读写权限
- 重新运行 `npm run seed` 初始化数据

### 问题4: 前端样式加载异常
**解决方案**:
- Tailwind CSS通过CDN加载，检查网络连接
- 清除浏览器缓存
- 重启前端开发服务器
