## 🔌 API接口用例演示

### 1. 书籍相关API

```bash
# 获取所有书籍
GET http://localhost:3001/api/books
Response: 7本书的完整信息，包含标签

# 获取单本书详情
GET http://localhost:3001/api/books/{bookId}

# 通过ISBN添加新书
POST http://localhost:3001/api/books/isbn/9787020007852
Response: 自动获取的书籍信息

# 更新书籍状态
PATCH http://localhost:3001/api/books/{bookId}/status
Content-Type: application/json
{ "status": "reading" }

# 更新阅读进度
PATCH http://localhost:3001/api/books/{bookId}/progress
Content-Type: application/json
{ "currentPage": 180 }
```

### 2. 阅读会话API

```bash
# 开始阅读会话
POST http://localhost:3001/api/reading/sessions
Content-Type: application/json
{ "bookId": "{bookId}", "startPage": 156 }
Response: { "id": "...", "startTime": "2024-05-16T..." }

# 结束阅读会话
PATCH http://localhost:3001/api/reading/sessions/{sessionId}/end
Content-Type: application/json
{ "endPage": 165 }
Response: 自动计算pagesRead、duration

# 获取书籍阅读统计
GET http://localhost:3001/api/reading/stats
```

### 3. 笔记API

```bash
# 获取书籍笔记
GET http://localhost:3001/api/books/{bookId}/notes

# 创建笔记
POST http://localhost:3001/api/notes
Content-Type: application/json
{
  "bookId": "{bookId}",
  "type": "highlight",
  "content": "这是一段精彩的摘录",
  "pageNumber": 123,
  "chapter": "第五章",
  "topics": ["孤独", "人生"]
}

# 按主题获取所有笔记
GET http://localhost:3001/api/topics/孤独/notes
Response: 跨书籍的同主题笔记聚合
```

### 4. 书单API

```bash
# 获取所有书单
GET http://localhost:3001/api/reading-lists

# 创建书单
POST http://localhost:3001/api/reading-lists
Content-Type: application/json
{
  "name": "治愈系书籍",
  "description": "在疲惫时能给你力量的书籍合集",
  "isPublic": true
}

# 向书单添加书籍
POST http://localhost:3001/api/reading-lists/{listId}/books/{bookId}

# 生成分享token
POST http://localhost:3001/api/reading-lists/{listId}/regenerate-token
Response: { "shareToken": "abc123xyz" }

# 访问公开书单
GET http://localhost:3001/api/shared-lists/{token}
```

### 5. 年度报告API

```bash
# 生成年度阅读报告
GET http://localhost:3001/api/reading/annual-report/2024
Response: 完整的年度统计数据
```

## 💡 特色功能亮点

### 1. 智能阅读速度预测
- 自动追踪每次阅读的页数和时长
- 计算个人平均阅读速度
- 根据剩余页数预测读完所需时间
- 可用于制定阅读计划

### 2. 跨书籍主题聚合
- 打破单本书的局限
- 将所有书中涉及相同主题的笔记聚合展示
- 支持横向对比和深度思考
- 构建个人知识图谱

### 3. Open Library深度集成
- ISBN一键获取完整书籍元数据
- 自动下载高质量封面图片
- 基于已读书籍的相似推荐
- 扩展发现更多感兴趣的书籍

### 4. 书单社交分享
- 一键生成公开分享链接
- 他人可无需注册查看书单内容
- 支持一键导入他人书单到自己书库
- 可追踪书单被查看次数

| 问题 | 解决方法 |
|------|----------|
